# Datasets para análisis de datos

Este repositorio contiene tres datasets para análisis exploratorio y predictivo.

## 1. TMDb Movies
**Contexto:** Información de películas.  
**Variables interesantes:** budget, revenue, popularity, genres, release_date.

## 2. Bank Marketing
**Contexto:** Campañas de marketing telefónico de un banco.  
**Variables interesantes:** age, job, marital, balance, y.

## 3. House Price Prediction
**Contexto:** Precios de viviendas en Chennai, India.  
**Variables interesantes:** size, bedrooms, bathrooms, location, price, year_built.
